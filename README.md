# AutoReview server

##Описание проекта
Система для отправки на линию машин.  
Серверная часть.

##Используемые технологии
[C++14](https://ru.wikipedia.org/wiki/C%2B%2B14)  
[Qt](https://ru.wikipedia.org/wiki/Qt)

##Системные требования
* Windows; Linux
* nginx
* PostgreSql v9.4 и выше
* php
* apache
* Qt5.5 >
